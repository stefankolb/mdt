Media Feature - (device-)width/height

This tool helps you to see whether or not the device supports the (device-)width/height media feature.

Custom

If you want to test for a specific width/height, enter horizontal and vertical resolution in pixels and watch the results.


If you want to test for a specific device-width/-height, enter horizontal and vertical resolution in pixels and watch the results.

Defaults

Default breakpoints for (device-)width are: 320px, 480px, 768px, 1024px, 1200px and 1600px.
Default breakpoints for (device-)height are: 320px, 480px, 720px, 960px, 1080px, 1200px and 1440px.
